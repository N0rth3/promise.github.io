闲言碎语

发表于 2019-03-03 | | 本文总阅读量

一时想不起来取什么标题，那干脆就随便取一个

今天偶然看到刘学长的公众号推送了
标题是读博之后的500天，发现原来一晃时间已经过去那么久了
关注他公众号的时候还是大一，那时候只知道他很厉害，保研去了清华。

后来他的公众号再也没有动静了，其实我也一样，写东西的频率越来越低。

青春梦想爱情都是奢侈品，直到失去了才惋惜。

其实大家都不容易。

好像刘学长在清华也郁郁不得志，paper被拒了又拒。
大概这也就是人生，苦中作乐。

也有恍然如梦。
我入门的时候看的就是米斯特的教程，不出意外估计还是key讲的。
今天我加入到了米斯特安全团队，key帮我推荐的。

人生啊不在于你完成了什么，而在于你做了什么。
就像刘学长这种一个月能被拒十次paper的经历，也定然是宝贵的。

所以何必患得患失，很多时候想得太多根本就不好。
埋头去做，能在一个领域混得风生水起的人，没有一个是容易的。

今天她学vb，我老和她说计算机是一片海洋，她说不是的，在她看来就是一片沼泽。
其实这多像我们的人生啊
其实用沼泽来形容计算机，也根本不为过。
抽象难懂，每前进一步都要付出巨大的努力。

但是呢，我感觉我不是在沼泽而是在海洋。

所以埋头去做，不要想那么多。

她又生气了啊，我不知道怎么办，因为我觉得不可理喻。

和她打电话的时候我妈说她懒，（我妈不知道我在打电话），我也没放心上，就给她说我妈说她懒。

她有情绪了啊，也不跟我吵，直接挂我电话。

然后又说以后不会去我家了，还说不是开玩笑。

那我能说什么呢，我很难过的，我也不知道找谁说，我也想不明白。

难道我不和她说我妈说她懒，那这个事情就不存在了嘛？不可能的。

所以我就是直男，把所有人都想得和我一样透彻。但事实不是这样。

我很不能理解她解决问题的方式，我妈说她懒她就不来我家，而不是说自己变得勤快一点。

为什么人就一点指责都不能接受呢？她说我可能不知道我妈这么说她有多难受。

我确实不知道我说。

因为在我看来大人说小孩子懒是特别正常的，况且我妈说的也确实是事实啊。

所以接受一点小的指责就难过，然后给我说不来我家。

那我也挺难过的。

就是这么任性啊，宁愿不来我家也不愿意说下次勤快一点，让我妈下次夸夸她。

既然都不来我家，那还结什么婚呢，所以她大概也不知道她说这些话我有多难过。

她说我总把事情想得简单，是的，我确实把事情想得很简单。

闹脾气就挂电话，然后就做很任性的决定。

我也不知道怎么去引导她，我很爱她，也想和她过一辈子。

然后她今天给我说感觉我们最后走不到一起。

我嘴上不说其实我很难过的。

我很长一段时间习惯了什么都给她说，现在我突然想起来我原来还有博客这么个地方可以畅所欲言。

我觉得我就只差把心掏出来给她，可是她很多时候都不体谅我。

就是个小朋友啊，所以也就是我欠她的。

我也想我有脾气就直接挂电话，然后什么都不管倒头哭一通然后睡觉，可是我做不到，因为这样太不负责任。

爱一个人何必这么折磨他，在我看来分明很简单的恋爱和生活。

单今天的事讲，本身也不是什么大事，何必上升到不来我家。

逃避真的就那么根深蒂固嘛。

可能我永远想的都是改变。

其实我就是那种撞了南墙也要把南墙撞倒的人。

以前我以为自己无所不能，有注定属于自己的光彩。

后来发现有的事你拼尽全力也办不到。

然后我明白很多事即便办不到也要去拼而不是逃避，因为你收获的可能是另一种可能。

我也一直在引导她，可有的时候我真的很迷茫，我不知道我要怎么做。

在我看来每天都是充满希望的一天，未来有那么多的美好等着我，我有很多很多的事情要去做。

唉，也不知道怎么说，写了这么多心里虽然还是难过，但多少好了一点。

我给她说我突然很害怕。

我爱她嘛，很爱很爱，深入骨髓。

但是我脑海里突然跳出一个人，云舒。

我在想是什么样的情况让他做出永不再婚的决定呢。

我突然对婚姻产生了一丝恐惧。

其实人生就是一场又一场的豪赌。

每个时刻我都愿意相信自己的判断。

不知道怎么弄，可能明天她就知道自己不对了吧。

当然，可能还是不知道。

那我也不知道。

我也有好多事情要做的。

反正我是爱她的，也是想好好过的。

无论是考研还是工作，我都在被推着走。

生活不易，每个人其实都不轻松。
