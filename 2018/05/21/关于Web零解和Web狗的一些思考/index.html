关于Web零解和Web狗的一些思考

发表于 2018-05-21 | | 本文总阅读量

Rctf2018结束了，我们队的Web最终零解，楼下复旦的队伍Web ak，虽然说没想过要ak，但是零解还是很不是滋味


即便两天的比赛很是划水，但是也明白自己现在这个水平，要做这种Web题，也确实是太勉强了
国赛的时候放了两个Web，一个签到一个压轴，那个时候感叹Web狗真的是无法存活
这次Rctf放了7个Web，应该说都是高质量的题目，然后我零解，确实应该反思
yype和我讲还不知道17届怎么办
想了想17届最强的晴晴不打比赛，除了AiHimmel做一做misc，确实没人在打比赛
那下一届的出路又在哪里，我也给不出答案

写写Web狗的看法吧
前几天看一个dalao的博客，突然明白一些道理。队友这两个字，来得远比同学这两个字重多了。CNSS的荣誉是一届又一届传承下来的，它的成绩是大家共同努力得来的。我很明白自己什么水平，知道自己不可能说能把CNSS送上世界舞台，取得什么辉煌的成绩，但是也不能让它在我这一届就这么没落下去。所以今天看到Web零解的时候，其实挺沉重的。你有多强，就配拥有多强的队友。
但是今天我们pwn和密码学都ak了，Web零解，所以我很羞愧。我的队友都很强，是我拖了后腿。我想变强，非常的想，加入凝聚这么久以来从未有过如此迫切的想法。早上起床看到yype给我找到的web入口，我看了一个小时，有些思路，感觉是能做的。但是代码审计的基本功实在是太烂，我很讨厌这种无力感。在赛场上时，你的队友只能选择无条件的相信你，yype给我找到了入口，可惜我没能做出来。和国赛的时候同样乏力，国赛的时候是Web太少，这次是Web太多。
滴滴的面试也让我想到了很多东西，其实Web有多高的水平它不应该是零散化的，它需要系统的学习，而不是你刷过多少题目。这些优质的Web题，都不再是套的模板，都需要扎实的基本功。换句话说，多少付出，多少收获，不是说平时考得少的东西就不用学，更不是说只用去学习各种各样的漏洞。Web安全的学习不应该是零散的，它是一张网，网的深度就决定了你的实力，横向纵向。
话题回到凝聚来，我很庆幸自己能加入凝聚。因为和我的队友们比起来，我一直都是个菜鸡。不过从大二加入凝聚以来，我的水平虽然还是最菜的，但是进步绝对是最大的。我无数次的回想起大一wind面我的时候，他问我的傻逼问题，我并没有答上来。我也在想如果我大二开始的时候选择了放弃，那现在是什么情况。我觉得我的人生有两个重要的转折点，一是高考，二就是加入凝聚。如果没有加入凝聚的话我现在肯定是条咸鱼，最后大概也不会做安全，更不会去面什么滴滴安全部。也就和大多数你电学生一样，上课打游戏。也不挂科，也不写代码，反正就是混。
因为加入凝聚所以我才得以融入安全圈，认识越来越多的大佬，被一次又一次的刷新认知，然后也以此为不断前行的动力。也因为CNSS这四个字母的缘故，出门在外也有了归属。UESTC和CNSS，算是出门在外的寄托了。也穿着CNSS的衣服晃来晃去，用着队友们打比赛拿来的奖金吃着火锅，喝着队友们请的奶茶，然后被狗头爸爸等一系列学长教育和鼓励。但越是这样，我就越是愧疚。因为还是那句话，你有多强，你就配拥有多强的队友。我没有划水的理由，不然我凭什么享受那么多东西，队友的信任，帮助，鼓励。
今天yype给我讲杜姐姐让他做下一届负责人的时候我才发现原来又要换届了。想起来当初方姐姐面试时问我的问题，你为什要加入凝聚，你觉得你能给凝聚带来什么。我说加入凝聚的初衷就是学习技术，我热爱凝聚的氛围。那我能给凝聚带来什么呢，我的答案是希望我能将凝聚这种氛围不断的传承下去，也希望不断的提升自己，在队友们需要我的时候，我能够站出来。
那现在的状况是这样，大三没有Web手，大二我，乖乖姐姐还有兰州，大一止痒同学。兰州自从恋爱以来大概就没怎么打过比赛，他放不下的太多，舞社那么多事，而且他又是学霸。止痒看得出来也很认真，但是他的学习路线上好像并没有规划大二要打比赛这一点，感觉比赛中都没有见过他。乖乖是当之无愧的现任Web最强，即便很不想承认，但确实他除外就剩我了。美工只能算半个Web，全栈选手。这两天的比赛兰州和止痒都没打，就我和乖乖在做(虽然我也只是看了看题没做出来，也就是我所说的，队友需要我的时候。可是我什么也不能做，从小到大，我都很讨厌这种乏力感。
突然想明白为什么自己那么喜欢看海贼王，不仅因为它很真实，每个人都有自己的性格。更因为他强调了伙伴，我的队友就是我的伙伴，这里没有勾心斗角，没有利益关系，大家为了一个目标，共同努力。索隆被鹰眼打败后，撕心裂肺的吼出那句，我绝对不会再输了。但现实就是他之后依旧被和平主义者和黄猿按在地上摩擦，路飞让大家快逃的时候是多么的乏力，他们都在想自己不会再输了吧。所以之后的两年里，索隆会舍弃尊严拜鹰眼为师，因为他要守护重要的人。尽管说起来很中二，但确实就是这样，我对于CNSS的感情是很深的，我也想要守护它，我也绝对不会再输了，我要看到我的队友走到更远的地方。为了CTF我可以放弃很多东西，因为我还在这个队伍里，我就不能忘记我加入CNSS的初心，所以我能够放下摄影，放下电竞，一心一意的打比赛。
我绝对不会再输了！
