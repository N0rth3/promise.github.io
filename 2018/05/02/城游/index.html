城游

发表于 2018-05-02 | | 本文总阅读量

小时候，总想知道沙漠那边是什么。
走过去，才知道沙漠那边，还是沙漠。

这是东邪西毒里欧阳锋的台词。

人生第一次出远门，1800公里。

那个曾经的少年也走到很远很远的地方了。

这么多年来其实一直向往着北京，不仅因为江南的书中提到的那句，北方有佳人。更多的是我对北京的了解，来自江南的书中，其次大概是中国合伙人。

那个被江南描述的无比美好的地方，北大。所以我也幻想清华园里也是同样的光景，北大清华，中国最顶尖的两座学府，有着太多的光环，那里有太多传奇的人物，所以我会无比的向往。可惜考不上，没能结识那么多风云人物。

所以到北京最想去的反而是北大清华，但当被清华拒之门外要求出示学生证时，是深深的挫败。很多你无比向往的地方，你终究是到达不了，哪怕只是作为一个过客都被种种限制，我和那些在外面和清华大学四个字合影的游客没有什么区别，不属于这里。

和北京这座城市给我的感觉是一样的，我不属于北京。在北京的几天里，我无比的怀念成都，第一次像一个离家的游子。成都的血骨已经融入到我的灵魂，它让我和北京这座城市格格不入，我不喜欢北京。不喜欢可能在于很多方面。

味道肯定是排在首位的，北京的辣很假，我不知道它假在哪里，即便是那天吃过的小炒黄牛肉，它很辣，比成都的很多菜都辣，但就是辣得不真实，感觉辣在表面，并不好吃。北京的米饭是论碗卖的，一碗不够又要买新的一碗。吃不惯北京羊肉里的那种膻味，即便知乎上强烈推荐的小吃店，也很是勉强。北京很多菜甜得过分，而我又是向来不喜甜为主食的。在潇湘府吃的久负盛名的北京烤鸭，各个方面都无可挑剔，但仍然差一种感觉，它给我的可能还没有寝室楼下加蛋加肉的炒面来得实在。

北京的司机好像都不喜欢闲聊，尽管他们也很热心，送我到机场时也会给我说怎么怎么走。这里没有路边摊的烧烤摊，也没有夜啤酒，晚上十一点的中关村就基本看不到人了。但飞机刚落地来到成都的时候，接我们的司机和我一路聊到学校，想起重庆，重庆的司机从我上车就滔滔不绝，疯狂给我推荐哪里好玩，大概这也是为啥去重庆的时候没有什么感触的原因吧，重庆和成都不过是一脉相承。

但我以后还是会来这里实习，来这里工作，不过即便那个时候，我还是不会喜欢上这里。但那个时候我是属于这里的，我会慢慢的习惯这种不喜欢，以一种我喜欢的方式生活。最后我的江湖依旧会是成都那座城市，很多年前我不能理解江南说的那句话，路的尽头会有一座城市等你。现在我理解了，我路的尽头是成都，在北京的这几天，我很怀念成都。

在寝室楼下下车的时候，接近十二点，熟悉的炒面师傅，加蛋加肉加辣椒。师傅说感觉好久没看见我了，问我去哪里了，我说刚从北京回来，吃着他给加的榨菜，于是我很满足，在这座血脉相连的城市。

不过像龙与少年游里讲的那样，我始终会去到很远很远的地方，虽然我明白我路的终点是成都。

Gai唱的很好，穷人的头啊你莫望到天，行万里路啊读万卷书。这趟来北京，收获之大是我不曾想过的。打比赛的时候没想过会被邀请，曾几何时，我也值得别人用诚邀莅临这四个字了，当年连一件CNSS的T恤都拿不到的人，现在已经有人给报销所有费用邀请去北京。认识了很多dalao，越发觉得自己辣鸡，为了对得起诚邀莅临这四个字，对得起CNSS，对得起UESTC，对得起我头上的ID，我不能这么菜，这就是不断学习的动力吧。

夜影能登顶也是有道理的，看着他博客每天都会写的总结，持续一年多，其实差距就是一点一点出来的。yype才大一就能做那么难的题目，欧嘉俊神仙就更不用多提。而我呢，web又菜，密码学又没学好，杂项也只能看运气，人与人的差距真的挺大的。不过也没什么好气垒的，其实我真正接触安全而不是成天划水也不过是半学期的事情，自己付出了多少时间自己心里还是有点逼数的，这半学期的成长我已经很满意。当一名赛棍，这就是我今年的目标，我相信明年的国赛，即便仍然只有两个web,我也不会像今年这样无奈。也明白了数学的重要性，痛下决心好好学习数学，说不准也会考研。

不过在那之前，我要做一名赛棍。

来滴滴的时候改了ID,我要让N0rth3ty这个ID在圈子闯出一片天地。

在北京打国赛的时候发现自己还是给不到队伍更多的帮助，yype爆肝的时候，我也只能喊666，一方面是出题的原因，但不可否认的也是，确实是自己太菜。不过我做出那道流量分析的时候，却是感慨万千。Wind在群里称赞了我一句，结果带了一波节奏，直接刷屏了。

曾几何时，我也值得wind称赞了。想起来大一的时候wind是面试官，问我header是什么，怎们看。我答不知道，我做题就是用直接用的工具，然后没有通过CNSS的面试。那个时候觉得自己是萌新，也没觉得自己有很菜，想不通为啥不让我过，现在想来我如果是当时的面试官估计很无语。

大概wind已经忘记了这件事，也压根不知道他当年还面试过我，从North到N0rth3ty，一晃两年了。现在我也是CNSS的一员，不过国赛看到我们总榜第二的时候我依旧帮不上忙，最后半小时被上交无情反超，很是遗憾。

从始至终，我都很讨厌这种乏力感。

不忘初心。
