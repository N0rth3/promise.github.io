电竞之星

发表于 2017-03-26 | 分类于 随笔 | | 本文总阅读量

如果没有复活赛的话，那可能就是一轮游了，很难受，bo1翻车。
第一次出任队长，虽然没想过夺冠，但还是想取得好一点的成绩。赛后曾经的队长给我讲了很多，反思了下，我的问题确实很大。
第一次发现lol这个游戏沟通原来这么重要，也第一次认识到队长这个位置是这么的重要。
第一场比赛没人说话的时候我没有站出来去猜拳选蓝紫色，我想我么这么强也无所谓吧，对面也不会选杰斯的，结果对面一选杰斯。
看着对面是从来没听过的队伍就没放在心上，bp也太过随意。看到锤石杰斯冰女我还是选了大嘴，没有考虑过逆风的问题，总感觉对面不强。
队伍逆风的时候安静得可怕，看着队友被拿了一血，大家都不说话，我也保持沉默。
比赛打得没有中心，没有人指挥，每个人都在打自己的，连正常发挥都做不到。比分0-6的时候，比赛其实就已经结束了，我从头到位都没站出来过，没有组织过一波团战。
比赛中我几乎没有开过口，第二把老队长打中单的时候情况明显不同，队伍开局有过小逆风，但始终他在指挥，我们的目的很明确，那个时候我们才像一只队伍。
第一把输了大家也不说话，我还是选择了沉默。有的时候沉默太可怕，身为队长的职责就是让一支队伍运作起来，有自己的战略，而不是一盘散沙。
今天即便我们能打出打rank的水平也能获胜，可是打得太过压抑，阵容压制，局势也不行，连高分路人的水平都没有打出来。像rank一样去打一场比赛的话，那大概就是今天这钟结局。
上次参加比赛，逆风的时候队长会鼓励，会指挥，会给我安排训练赛，会告诉我练什么英雄。
但这次我身为队长从始至终没有研究过bp，也没有想过我们要选怎样的阵容，把初赛当训练赛打，也没有组织去比赛的场地打打训练赛适应适应机器，我的问题很大，我很抱歉。
如果有复活赛的话，我们再尽力争取，如果没有了，那我们还有不少比赛可以打嘛。
