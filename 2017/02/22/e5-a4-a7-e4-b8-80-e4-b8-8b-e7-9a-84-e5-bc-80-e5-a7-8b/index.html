大一下的开始

发表于 2017-02-22 | 分类于 随笔 | | 本文总阅读量

新学期再次早起，赶着去教室上早自习。想来对早自习是又爱又恨，也只能认命。
这学期没有晚自习了，感觉跳舞的动力也少了很多，没有晚自习就不想九点去品学ab4了，有点惰怠。说起来舞社的人我也不熟，周六说大家分享特产，可是我什么都没带，在考虑去不去。晚上的时间独立安排，一下感觉有很多的时间，看小说也好，写代码也好，杂七杂八。
这学期不准备怎么折腾了，学生会的工作没什么意义，那就混混学分吧。想做设计师什么的也放一放，讲起来我已经缺了why not工作室的一次考核了。
摄影的话，有机会就拍拍吧，不会去拍什么晚会了，没什么意思，有机会的话想出去旅行。
这学期准备去做做家教，将我关于高考的感悟传授一番，讲道理我理综还是很强的。然后准备参加支教队，可以的话暑假我想去支教，做一些有意义的事情。
昨天第一次利用了漏洞提前查到了四级成绩，挺有成就感的，不过这周末就是我的线代补考，想来还没有开始复习，有点烦。
这学期的课很多，数据结构，微积分，数字逻辑，大物，离散等等，挂科很难受，不想再挂科了。
所以早睡早起，没必要的话不会熬夜了。
一切也就那样，平平淡淡。这学期过了就要搬沙河了，圈子可能会更小吧，也不知道这学期又会认识怎样的人。前路未知，不过这样也好。
不再那么浮躁，默默的努力，去成为更加优秀的自己。
