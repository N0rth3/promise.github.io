沙河第一天——又是新的开始

发表于 2017-07-08 | 分类于 随笔 | | 本文总阅读量

昨晚和老队长坐在清水河主楼门口喝酒，挺感慨的，但依旧觉得没什么好伤感。
昨晚迷迷糊糊基本没睡着，今天四点起床开始搬东西，坐在校车上慢慢驶出清水河的时候，清水河还是留下了太多回忆，虽然不是一去不回，但多少还是有点伤感。下车看到墨鱼来接我，然后帮我搬东西的时候，恍惚感觉回到了高中，命运这种东西，谁说得清楚呢，可笑的是我不信命。
如果没去过南山的话，我可能也会对新宿舍很不满意。我们估计搬的是最老的一批寝室，确实是很破旧了。看到周围很多人在吐槽，其实我觉得也不是那么夸张，反正我觉得还行，随遇而安嘛。就像刚去南山发现要用饭盒吃饭，发现厕所没有门，其实后来也就习惯了。有德时候觉得他们就是娇生惯养，好多人已经在商量租房住了，真的有这个必要嘛？我也只能说有钱真好。
昨晚跟老队长聊了很多，像他说的那样吧，在哪都一样，好好干，何况沙河还有墨鱼在。
今天是在沙河的第一天，开始一段新的旅程，同我跟墨鱼碰杯时讲的一样，新的开始。
一年过去了，虽然同最开始的规划有些不同，然后上学期立的flag也倒了一些，不过不后悔吧，大一的生活丰富多彩，挺折腾的，像我之前讲的一样，大一生活我打90分吧。
唉，本来是想总结一番的，不过我大概是越来越不能写了，想了半天，发现也就那样，没什么好写的。忽然又想的江南的话，“人就是这样，小时候爱唱歌，唱破了喉咙也没人听懂，如今有人想听了，你却不想唱了。”不过大概到今天想听的人也不多。我也懒得对很多事情发表评论，最多也就挂在这里，熟悉的人看看。
慢慢和傻逼心平气和的说话，然后接受现实，反正我只管自己和我在乎的人，我也不知道我什么时候这么自私了。以前觉得别人什么地方有问题还会提出来，现在是关我什么事，你该混混，爱咋咋地，我哪里那么多精力去关注无关紧要的人，更别提去生气，何必自找烦恼。
一路走来，大家都在变，但有的感情是变不了的，今晚跟墨鱼喝酒的时候，我觉得我们跟当年还是一样的，没有什么区别。一路走来，谢谢好多人给我的帮助。
能看到这篇文章的，肯定有瓜皮队长，墨鱼，光军，都挺感谢你们的吧，我平时倒是不会说这种话。
矫情的句子也懒得写了。
讨厌平庸的自己，我就是想赢啊。
管他什么新的开始，努力做更优秀的自己。

生活如歌
愿无岁月可回头
