愿无岁月可回头

发表于 2017-07-29 | 分类于 随笔 | | 本文总阅读量

瓜皮队长问我最近都在干什么，分也不上了，说说也不发，博客也不写。其实我也在想，放假这一个多月，我在干什么。不过最近确实也有很多事情吧。
和光军加一个同学参加了“绿盟杯”，也是第一次组队参加比较正规的ctf比赛吧，距离我接触ctf，时隔一年。比赛打得还行，虽然最后叫了叶姐姐来救场，不过从最后排名来看，没有救场还是能进决赛的。不过这都不是重点，重点是发现差距真的很大，问叶姐姐一个逆向中的代码审计的时候，发现自己要写了十多分钟的代码，在叶姐姐那里不过是几行python。然后另一个一起打比赛的同学，他入门比我晚，但一直在认真学习，这次能进决赛，他做的那个500分的解密很关键。回过头看看我自己，虽然队伍配置不合理，只能我硬着头皮怼逆向，但题目我也看过，我知道我即便做web，做杂项，我会做的，他们都会。除了逆向的那两个签到题，我大概是全程划水，觉得自己真的挺菜的。又适逢凝聚新一年的招新，仅管只是个暑期练手题， 发现web依旧不能ak，几分感慨，又想起去年。去年我乍一看分数很高，web题也做得差不多了，最后被刷，刚开始还有点不服气，现在看来被刷是肯定的。其它的具体细节我不太记得了，我只知道wind问我header是什么，怎么看的时候，我答不上来，因为我做那个题的时候就是百度的怎么改header，然后用modify header改的，但我并不明白后面的本质，只是在做题罢了。连header是什么怎么看都答不上来，那还做什么web呢？我很想很想加入凝聚，因为我是真的热爱安全，真的喜欢一起打ctf的那种感觉。然后发现仍旧是个菜鸡，也问过自己原因，无非是想得太多，做得太少。
再想想自己放假这么久的状态，每天是有在学，不过一天也就两三个小时罢了，所以忽然会害怕，害怕又被凝聚拒之门外。之前给清月写了一封信，她倒是回了我微信，不过信还没回，也不知道是个什么态度。不过看她去了牛津交流，再看看空间一堆出国交流的同学，落差怎么还是有的。我不考研，也不出国，那我凭什么比他们优秀？比他们混得更好，我觉得没有其它的答案，只能是努力和热爱。可惜想做的事情太多，最后一件也做不好。大一一年做了很多事情，学到很多东西，也确实有很大的进步和改变，在各向能力上都有长足的进步，但是这不包括技术，依旧是个菜鸡，也许在不做技术的人眼中还行，然而在大巴这群人中，我确实是垫底的一波。
我很明白自己现在该干什么，可是就是拿不出魄力。眼看还有几天就要去支教，我忽然明白留给我的时间不多了，我不能这么懒懒散散的学下去。我得拼一波，因为今年面我的标准，已经不是小白，我是大二的人了，破釜沉舟，这是加入凝聚最后一次机会。
只希望看到面试结果的时候，不像去年那样遗憾。
愿无岁月可回头。

沙河第一天——又是新的开始
